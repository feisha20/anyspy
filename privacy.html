Privacy Policy for AnySpy

Permissions Usage

activeTab Permission

AnySpy requires the "activeTab" permission for the following reasons:

  • To allow the extension to interact with the current webpage when activated by the user
  • To identify and locate web elements when users hover over them
  • To generate accurate XPath, CSS, and ID selectors for the selected elements

Data Collection

AnySpy does not:

  • Collect any personal information
  • Store any user data
  • Send any information to external servers

Usage

The extension only activates when explicitly triggered by the user through:

  • Clicking the extension icon
  • Using the keyboard shortcut (Alt+Shift+S)
